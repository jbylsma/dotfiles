* Extend Windows thumb key mode
* Home     | MetaL (Windows key)
* PageUp   | MetaR (Windows key)
* WindowsR | AltR
* End      | PageUp
[home]>[lwin]
[pup]>[rwin]
[rwin]>[ralt]
[end]>[pup]

* Use Vim vertical arrow keys ordering
[up]>[down]
[down]>[up]

* Left pipe to insert
[intl-\]>[kp-insert]

* Escape Keypad with Capslock
[kp-caps]>[kptoggle]

* Bind audio controls in Keypad mode
[kp-F4]>[prev]
[kp-F5]>[play]
[kp-F6]>[next]

* Bind display controls in Keypad mode
* Works for macOS
[kp-F1]>[scroll]
[kp-F2]>[pause]

* Swap audio controls into Normal mode
* Scroll seems to save intermittently. Try resaving by itself.
[scroll]>[vol-]
[pause]>[vol+]
[prtscr]>[mute]
[vol-]>[scroll]
[vol+]>[pause]
[mute]>[prtscr]
